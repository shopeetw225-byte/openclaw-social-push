## 发布微博 workflow

1. 确保当前标签页已由 OpenClaw Browser Relay 扩展附着，然后导航到微博主页：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay navigate "https://weibo.com"`
2. 等待页面初步稳定：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay wait --time 3000`
3. 获取交互快照：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay snapshot --efficient --interactive --compact`
4. 点击内容输入框：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay click <内容输入框ref>`
5. 输入微博内容：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay type <内容输入框ref> "{微博内容}"`
6. 如果有图片，优先用文件 input ref 或上传按钮 ref 上传 staged 文件：
   - `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay upload --input-ref <文件输入ref> "/tmp/openclaw/uploads/<图片文件>"`
   - 或 `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay upload --ref <上传按钮ref> "/tmp/openclaw/uploads/<图片文件>"`
7. 上传后等待并重新快照：
   - `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay wait --time 2000`
   - `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay snapshot --efficient --interactive --compact`
8. 点击发送按钮：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay click <发送按钮ref>`
9. 发布后确认：
   - `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay wait --time 1500`
   - `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay snapshot --efficient --interactive --compact`

## 注意事项

- **微博字数限制**：普通用户 140 字，会员用户 2000 字
- **图片支持格式**：JPG、PNG、GIF、HEIF、HEIC，最多 9 张图片
- **视频支持格式**：MP4、M4V、MKV、FLV
- **话题标签格式**：`#话题内容#` （前后都需要 # 号）
- **元素 ref 变化**：每次操作后都用最新快照重新确认 ref
- **首页长连接较多**：优先使用短等待 + 快照，不要强依赖 `networkidle`
- **如果看到“登录/注册”**：说明你的常用浏览器当前没有登录微博，先登录再继续
- **无草稿功能**：微博的简单发布框没有保存草稿功能，只能通过不点击发送按钮来保留内容

## 常见问题

1. **发送按钮不可点**：先检查是否仍停留在登录页，或图片仍在处理中
2. **上传失败**：检查 staged 文件是否还在 `/tmp/openclaw/uploads`，单张图片建议不超过 5MB
3. **内容被限制**：微博有敏感词过滤，发布前建议用户自行检查
