## 上传图文 workflow

仅当用户明确要发“小红书图文”或提供了图片时使用本 workflow。
如果用户没有给图片，只给了文案，直接改走“小红书长文” workflow，不要停留在这个上传图片页。

1. 确保当前标签页已由 OpenClaw Browser Relay 扩展附着，然后导航到图文发布页：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay navigate "https://creator.xiaohongshu.com/publish/publish?source=official&from=tab_switch&target=image"`
2. 等待页面稳定：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay wait --time 3000`
3. 获取交互快照：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay snapshot --efficient --interactive --compact`
4. 上传 staged 图片：
   - 如果快照里能拿到 `<input type=file>` 的 ref：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay upload --input-ref <上传输入框ref> "/tmp/openclaw/uploads/<图片文件>"`
   - 如果只能拿到上传按钮 ref：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay upload --ref <上传按钮ref> "/tmp/openclaw/uploads/<图片文件>"`
5. 等待图片处理完成：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay wait --time 2000`
6. 重新获取快照：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay snapshot --efficient --interactive --compact`
7. 输入标题：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay type <标题输入框ref> "{标题}"`
8. 点击正文编辑区并输入正文：
   - `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay click <正文编辑区ref>`
   - `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay type <正文编辑区ref> "{正文内容}"`
9. 如果有话题，在正文区继续输入 `#{话题}`，再执行：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay press Enter`
10. 发布前重新快照：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay snapshot --efficient --interactive --compact`
11. 点击发布按钮：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay click <发布按钮ref>`
12. 发布后等待并确认：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay wait --time 1500`

# 注意
标题不要超过 20 字符
上传完成后如果发布按钮仍不可点，优先检查图片处理是否结束
ref 变化很快，标题、正文、发布按钮要按最新快照重新取值
