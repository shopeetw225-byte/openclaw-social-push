## 发布掘金文章 workflow

1. 确保当前标签页已由 OpenClaw Browser Relay 扩展附着，然后导航到掘金新建文章页：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay navigate "https://juejin.cn/editor/drafts/new?v=2"`
2. 等待页面稳定：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay wait --time 3000`
3. 获取交互快照：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay snapshot --efficient --interactive --compact`
4. 输入标题：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay type <标题输入框ref> "{标题}"`
5. 准备正文：
   - 如果正文来自 Markdown 文件：`pbcopy < "{正文文件路径}"`
   - 或 `printf '%s' "{正文内容}" | pbcopy`
6. 点击正文编辑区并粘贴：
   - `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay click <正文编辑区ref>`
   - `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay press "Meta+V"`
7. 等待自动保存：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay wait --time 2000`
8. 重新快照，确认页面已进入可发布状态：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay snapshot --efficient --interactive --compact`
9. 点击“发布文章”按钮：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay click <发布文章按钮ref>`
10. 如果出现发布设置面板，依次完成必填项：
    - 重新 `snapshot`
    - 选择分类 / 标签 / 封面
    - 如果要上传封面，优先使用 `upload --input-ref` 或 `upload --ref` 上传 staged 文件
11. 点击最终发布按钮：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay click <确认发布按钮ref>`
12. 发布后等待并确认：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay wait --time 1500`

## 注意事项

- 掘金正文编辑器更适合“聚焦后粘贴正文”，比逐字符输入稳定
- 点击“发布文章”后通常会进入发布设置弹层，所有分类/标签 ref 都要用最新快照重新拿
- 本 workflow 目标是最终发布，不是只保存草稿
