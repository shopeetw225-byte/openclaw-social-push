## 发布微信公众号文章 workflow

1. 确保当前标签页已由 OpenClaw Browser Relay 扩展附着，然后导航到公众号后台：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay navigate "https://mp.weixin.qq.com/"`
2. 等待页面稳定：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay wait --time 3000`
3. 如果当前不在文章编辑页，用稳定选择器打开“新建文章”：
   - `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay evaluate --fn '() => { const el = document.querySelector(".new-creation__menu-title"); if (el) { el.click(); return true; } return false; }'`
4. 等待编辑器加载：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay wait --time 3000`
5. 获取交互快照：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay snapshot --efficient --interactive --compact`
6. 输入标题：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay type <标题输入框ref> "{标题}"`
7. 如果要选择作者：
   - 点击作者框：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay click <作者输入框ref>`
   - 重新快照后点击作者候选项：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay click <作者候选ref>`
8. 准备正文：
   - 长文优先先放入剪贴板：`pbcopy < "{正文文件路径}"`
   - 或 `printf '%s' "{正文内容}" | pbcopy`
9. 点击正文编辑区并粘贴：
   - `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay click <正文编辑区ref>`
   - `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay press "Meta+V"`
10. 如果有封面，先打开封面选择入口：
    - `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay evaluate --fn '() => { const el = document.querySelector("#js_cover_null .js_imagedialog"); if (el) { el.click(); return true; } return false; }'`
11. 重新快照后用文件 input ref 或按钮 ref 上传 staged 封面：
    - `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay upload --input-ref <封面输入框ref> "/tmp/openclaw/uploads/<封面文件>"`
    - 或 `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay upload --ref <封面上传按钮ref> "/tmp/openclaw/uploads/<封面文件>"`
12. 等待上传并重新快照：
    - `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay wait --time 2000`
    - `/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay snapshot --efficient --interactive --compact`
13. 如果出现裁剪或下一步弹窗，依次点击 `<下一步ref>`、`<确认ref>`
14. 如果出现原创声明弹窗，按当前任务要求点击 `<声明原创ref>` 或 `<确认发布ref>`
15. 发布前重新快照：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay snapshot --efficient --interactive --compact`
16. 点击发表按钮：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay click <发表按钮ref>`
17. 发布后等待并确认：`/Users/openclawcn/.homebrew/bin/openclaw browser --browser-profile chrome-relay wait --time 1500`

## 注意事项

- 正文编辑器更适合先写入剪贴板再粘贴，不建议逐字符输入长文
- 封面上传优先用 `evaluate` 打开稳定 selector，再用 `upload --input-ref` 或 `upload --ref`
- 微信公众号经常出现二次确认弹窗，出现新弹窗后一定要重新 `snapshot`
- 此流程默认自动发布，不再停留在草稿态
